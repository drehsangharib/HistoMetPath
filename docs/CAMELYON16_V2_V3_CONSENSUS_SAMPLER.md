# CAMELYON16 v2/v3 Consensus Sampler

The development-only portfolio audit identified Spatial v2 and Spatial v3 as the Pareto front. The consensus sampler preserves coordinates shared by both methods, divides remaining capacity equally between v2-only and v3-only coordinates, deterministically reallocates unused capacity, and uses spatially dispersed selection without annotations.

Only training and validation coordinates are eligible. Annotations are used only after consensus construction for coverage comparison. Test slides and model outputs are prohibited, and the completed final test remains immutable.
